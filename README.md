# Customer Segmentation using K-Means Clustering

## Overview

This project applies the K-Means clustering algorithm to group customers into meaningful segments based on purchasing behavior and demographic information.

The project demonstrates how unsupervised machine learning can help businesses understand customer groups for better marketing and business decision-making.

---

## Objectives

- Import customer dataset
- Clean and preprocess data
- Perform Exploratory Data Analysis (EDA)
- Determine the optimal number of clusters
- Train a K-Means clustering model
- Visualize customer segments
